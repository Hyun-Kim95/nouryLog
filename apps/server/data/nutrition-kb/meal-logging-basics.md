# 식단 기록 시작하기

nouryLog에서는 아침·점심·저녁·간식 등 **끼니**를 고른 뒤 음식을 검색하거나 직접 입력해 식단을 기록합니다. 템플릿·최근 기록을 쓰면 같은 메뉴를 빠르게 반복 입력할 수 있습니다.

처음에는 하루 한 끼만이라도 꾸준히 남기는 것이 좋습니다. 매일 비슷한 시간에 알림을 켜 두면 잊지 않기 쉽습니다. 기록이 3~7일 쌓이면 주간 단백질·칼로리 요약과 AI 코치 인사이트가 의미 있게 보이기 시작합니다.

「식단 기록을 어떻게 시작하면 좋을까?」처럼 기록 방법을 물으면, 위 흐름대로 끼니 추가 → 음식 선택 → 저장 순서로 시작해 보세요. 구체적인 수치·목표 달성 여부는 기록이 쌓인 뒤 주간 통계 질문으로 확인할 수 있습니다.
