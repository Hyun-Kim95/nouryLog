# Google Play Data Safety — 보존·삭제 (v3 정합)

개인정보처리방침 v3(2026-06-20)와 동일하게 Play Console에 반영합니다.

| 항목 | 값 |
|---|---|
| 사용자가 삭제를 요청할 수 있음 | 예 (설정 > 회원 탈퇴, 식단 기록 삭제) |
| 서비스에서 즉시 비노출 | 예 |
| 완전 삭제까지 보존 | **1년** (비활성화 처리일 기준) |
| OCR 사진 | 분석 후 즉시 파기(1년 보존 제외) |
| 감사·운영 로그 | 5년(개인 DB purge 대상 아님) |

콘솔의 "데이터 삭제" / retention 관련 질문은 위 표와 [privacy.md](./privacy.md) §3을 기준으로 작성합니다.
