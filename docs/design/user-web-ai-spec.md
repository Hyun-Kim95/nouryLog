# user-web — AI 미리보기

- **네비:** 주간 리포트 · 월간 패턴 · AI 코치 (**「시연」 메뉴 없음**)
- **로그인:** `/demo` — **데모(시드)** 상단, **Google·Kakao·Naver SNS** 하단. SNS는 본인 기록, 데모는 시드 계정.
- **기준일:** AI 코치·주간 — 달력 모달(`AnchorDatePicker`). 월간 — 이전/다음 **±1달** 네비.
- **기본 경로:** `/` → `/ai/coach`

모바일이 제품 SSOT; user-web은 포트폴리오·로컬 API 확인용.
